# Actible

A standalone AI agent desktop application built for running, managing, and interacting with AI agents.

Actible brings AI agents into a dedicated workspace with tools, memory, and automation capabilities — all packaged into a single executable.

No complicated setup. No dependency installation. Just download, launch, and use.

## Features

* Single portable `.exe` application
* Modern desktop interface
* AI agent management
* Persistent conversations
* Custom agent instructions
* Tool usage
* File interaction
* Command execution
* Real-time responses
* Multiple model support
* Agent memory
* Workspace management

## What is Actible?

Actible is designed to make AI agents feel like real applications instead of a normal chat interface.

Agents can:

* Understand complex tasks
* Maintain conversation context
* Use tools
* Work with files
* Execute actions
* Complete multi-step workflows

The goal is to create a simple but powerful environment for interacting with capable AI agents.

## Installation

1. Download the latest Actible release.
2. Open `Actible.exe`.
3. Configure your AI model.
4. Start using your agent.

No installation required.

## Supported Models

Actible supports compatible AI backends, including:

* Local AI models
* Ollama
* llama.cpp
* OpenAI-compatible APIs
* Custom endpoints

## System Requirements

Minimum:

* Windows 10/11
* Modern CPU
* Internet connection (for API models)

Recommended:

* 16GB+ RAM
* GPU acceleration for local models
